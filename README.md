# Broker Proposal Generator

A simple real estate proposal generator for brokers, leasing professionals, landlords, and property advisors.

## Purpose

This project helps a broker enter client requirement details and property details, then generate a clean proposal workflow.

## Current MVP Features

1. Real estate proposal type selection
2. Client requirement form
3. Property detail form
4. Commercial terms section
5. Proposal preview / review flow
6. Broker-friendly workflow instead of generic agency packages

## Tech Stack

- HTML
- CSS
- JavaScript

## How To Run Locally

Open `index.html` directly or run a local server and open localhost.
