Before saving PDF

Final clean export checklist.

These small settings make the difference between an amateur export and a client-ready document.

  • Client PDF does not include WhatsApp or email drafts unless you intentionally add them as an appendix.
  • Browser print setting: turn off Headers and footers.
  • Turn on Background graphics if your PDF design uses accent backgrounds.
  • Review all numbers, property details, service charges, ownership documents, and assumptions before sharing.
  • Save one sample PDF and open it again to check page breaks before sending to a client.
