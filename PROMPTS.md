# Copy-Paste Codex Prompts

Use these prompts in Codex, Google Antigravity, VS Code, or another AI coding tool.

## Prompt 1: Explain The Repo

```text
Explain this repo like I am a real estate professional, not a developer. Tell me what each file does and how I can customize the app safely.
```

## Prompt 2: Residential Real Estate

```text
Customize this proposal builder for residential real estate agents. Add proposal types and blocks for apartment sale, villa sale, rental listing, buyer shortlist, and landlord pitch. Keep HTML, CSS, and JavaScript only.
```

## Prompt 3: Commercial Leasing

```text
Customize this proposal builder for commercial leasing brokers. Improve blocks for office leasing, retail leasing, warehouse leasing, rent, CAM, lock-in, fit-out, parking, possession, risks, and next steps.
```

## Prompt 4: Dubai Investment Sales

```text
Customize this proposal builder for Dubai investment sales. Use AED currency, Dubai market language, investment memo sections, service charges, title deed checks, parking allocation, rental yield, and exit potential.
```

## Prompt 5: Change PDF Design

```text
Change the client-facing PDF design style. Keep the app UI the same, but redesign only the proposal preview and print CSS. Make it feel like a premium real estate investment memo.
```

## Prompt 6: Add A New Proposal Block

```text
Add a new proposal block called Building Amenities. It should include lifts, parking, power backup, security, cafeteria, visitor access, and notes. Add it to Office Leasing Proposal and Landlord Property Pitch.
```

## Prompt 7: Prepare For Vercel

```text
Prepare this project for Vercel deployment as a static HTML/CSS/JavaScript site. Do not add a backend. Add any required config only if needed and explain deployment steps.
```

## Prompt 8: Fix Responsive Design

```text
Audit and fix responsive design issues in this app. Test desktop, tablet, mobile, and small mobile layouts. Keep the app simple and do not add new features.
```
